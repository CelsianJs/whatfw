# Changelog

## 0.1.0

- Initial release
- Syntax highlighting for What Framework JSX patterns:
  - Event modifiers (`on:click|preventDefault`)
  - Binding directives (`bind:value`)
  - Client directives (`client:load`, `client:idle`, `client:visible`)
  - Control flow components (`<For>`, `<Show>`, `<Switch>`, `<Match>`)
- Syntax highlighting for ThenJS APIs:
  - `defineConfig()`, `procedure`, `createRouter`, `createRPCClient`
  - Page exports (`export const page = { ... }`)
- 10+ What Framework snippets (`what-component`, `what-signal`, `what-effect`, etc.)
- 8+ ThenJS snippets (`then-config`, `then-page`, `then-api`, etc.)
- Language configuration for JSX/TSX bracket matching and comments
