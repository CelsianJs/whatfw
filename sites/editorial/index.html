01

Fine-Grained Signals

Reactive primitives that track exact dependencies. When a signal changes, only the code that reads it runs. No tree diffing. No wasted computation.

02

Islands Architecture

Ship zero JavaScript by default. Hydrate interactive components on idle, on visible, or on first interaction. The rest stays as static HTML.

03

Vanishing Footprint

The entire runtime weighs under 4kB gzipped. No virtual DOM overhead. No compiler required. Just JavaScript that runs in any browser.

The Technical Story

Code that reads like prose

No JSX compilation. No special syntax. Just pure JavaScript functions that return elements. Signals make it reactive. Islands make it efficient.

Start building → 
import { h, mount, signal } from 'what-framework';

function Counter() {
  const count = signal(0);

  return h('button', {
    onClick: () => count.set(c => c + 1)
  }, 'Count: ', () => count());
}

mount(h(Counter), '#app');
